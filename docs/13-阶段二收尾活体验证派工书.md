2. **灌知识库**（让 `knowledge_chunks` collection 有真实数据）—— 二选一：

     ```bash
     HF_HUB_OFFLINE=1 HF_ENDPOINT=https://hf-mirror.com OPENBLAS_NUM_THREADS=1 OMP_NUM_THREADS=1 \
     ```
