# ReflexLearn UniApp Demo

这是 ReflexLearn 的移动端录屏演示版，数据全部为本地假数据，不依赖后端接口。

## 页面

- `今日学习`：学习任务、Agent 调用过程、一句话推送视频。
- `知识库`：236 份知识文档、64 条用户需求、持续文档调用监控。
- `AI 辅导`：一句话诊断、薄弱点、视频和练习推送。
- `精品课程`：课程播放、章节进度、智能评论回复。
- `学习画像`：学习画像、行业差距、导师匹配。

## 运行

推荐方式：用 HBuilderX 导入 `uniapp` 目录，运行到浏览器、微信小程序或手机模拟器即可。

如果使用命令行，需要先在 `uniapp` 目录安装依赖：

```bash
npm install
npm run dev:h5
```

当前版本只做录屏展示，不需要配置接口和密钥。
